# AirHaven — Air Quality Monitoring System

AirHaven is a cross-platform IoT-based air quality monitoring system that provides real-time AQI data, automated alerts, and ML-based pollution forecasting. Built with React Native and Firebase, it transforms real-time environmental data into clear, actionable insights with an intuitive interface.

## Tech Stack

- **Frontend**: React Native, Expo, JavaScript/ES6+
- **Backend**: Firebase Realtime Database
- **Maps**: Leaflet (Android WebView), React Native Maps (iOS)
- **Build Tools**: Babel, Gradle, EAS Build

## Features

- Real-time AQI (Air Quality Index) monitoring from Firebase
- Interactive sensor node maps (Leaflet on Android, React Native Maps on iOS)
- Health recommendations based on current air quality
- Hourly and weekly air quality forecasts
- Detailed pollutant breakdown (PM2.5, PM10, O3, NO2, etc.)
- Animated visualizations and particle effects
- Multi-language support (i18n)
- Responsive design for various devices
